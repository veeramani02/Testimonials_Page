# Testimonials Grid Project

> Simple, responsive **Testimonials** section using CSS Grid .  
> Lightweight, easy to customize, and ready to drop into any frontend project.

## Features
- Responsive layout using **CSS Grid**
- Card-based testimonial items (name, role, avatar, quote)
- Easily configurable (number of columns, gap, card style)
- Works with  plain HTML/CSS
- Accessible markup (semantic HTML, alt text)

## Tech Stack
- HTML, CSS (CSS Grid + Flexbox)
